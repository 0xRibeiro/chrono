## Chrono - Password Security Analyzer

Chrono é um projeto simples desenvolvido com o objetivo de analisar a força de uma senha, avaliando seu nível de segurança contra possíveis ataques cibernéticos. A ferramenta estima o grau de proteção da senha e calcula quanto tempo ela levaria para ser quebrada por meio de ataques de força bruta (brute force).

Essas informações permitem que o usuário compreenda melhor a vulnerabilidade de suas senhas atuais e tome decisões mais conscientes para reforçar sua segurança digital.

O projeto foi idealizado como uma forma de aplicar, na prática, conhecimentos recentemente adquiridos em cibersegurança, além de aprimorar minha proficiência nas tecnologias e ferramentas utilizadas durante o desenvolvimento.

## Tecnologias Utilizadas

- Python – Backend e lógica de análise de senhas.
- Flask – Microframework web para o servidor e rotas da aplicação.
- JavaScript – Interatividade no frontend.
- HTML, CSS & Bootstrap – Estrutura, estilização e criação de interfaces responsivas.
